selenium basics practice tasks

Complete the steps on the main practice page using Selenium and verify visible text after each action. Back to practice page.

Step 1: Basic inputs and buttons

  1. Open seleniumbasics.html and locate the Username field, Password field, and Notes textarea.
  2. Type test values, for example: username student1, password pass123, notes first try.
  3. Click Submit and verify the Output text becomes exactly username=student1 | password=pass123 | notes=first try.
  4. Click Clear inputs and verify all 3 fields are empty and the Output text becomes cleared.
  5. Click Toggle submit enabled once and verify the Submit button is disabled and the Status text shows Submit is disabled.
  6. Click Toggle submit enabled again and verify the Submit button is enabled and the Status text shows Submit is enabled.
  7. Verify the Readonly field has readonly attribute and value readonly value.
  8. Verify the Disabled field is disabled and value is disabled value.

Step 2: Checkbox and radio

  1. Locate the Accept terms checkbox and its state text. Verify the initial state text is unchecked.
  2. Click the Accept terms checkbox, verify it is selected, and verify the state text becomes checked.
  3. Click the Accept terms checkbox again, verify it is not selected, and verify the state text becomes unchecked.
  4. Find all plan radio buttons (Basic, Pro, Enterprise) and verify there are 3 options.
  5. Click Basic and verify the selected plan text becomes basic.
  6. Click Pro and verify the selected plan text becomes pro.
  7. Click Enterprise and verify the selected plan text becomes enterprise.

Step 3: Dropdowns Select

  1. Locate the Country single select, the Skills multi-select, and the Selected output text. Verify initial output is country=none | skills=none.
  2. Select country value de (Germany) and verify output becomes country=de | skills=none.
  3. In the Skills multi-select, select selenium and sql. Verify output becomes country=de | skills=selenium,sql.
  4. Click Select defaults and verify output becomes country=lt | skills=selenium,junit.
  5. Click Clear selects and verify output returns to country=none | skills=none.

Step 4: Lists for findElements

  1. Locate the list area, the Count text, and the Last clicked text. Verify initial Count is 3 and Last clicked is none.
  2. Use findElements to collect all visible list items and verify the collection size is 3.
  3. Click the first existing item and verify the Last clicked text becomes Static item 1.
  4. Type My item into the New item text field, click Add item, and verify the Count text becomes 4.
  5. Find all items again and verify one item text is My item.
  6. Click Add 5 items and verify the Count text becomes 9.
  7. Click any generated item and verify the Last clicked text changes to that item text.
  8. Click Clear list and verify the list is empty, Count is 0, and Last clicked is none.
